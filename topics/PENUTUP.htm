FINAL ACCOUNT

 

1.      Tujuan dan Sasaran

Prosedur ini dipergunakan sebagai pedoman dalam mengevaluasi performance proyek melalui proses final account, sebagai dasar keputusan dalam pelaporan kinerja operasional perusahaan.

2.     Ruang Lingkup

Prosedur ini berlaku pada saat progress 100%, tutup tahun anggaran dan proyek closing  atau setelah  terlaksananya serah terima pekerjaan tahap ke 2 (dua) kepada pihak Pemberi Tugas / Owner.                 

3.     Referensi

3.3.     Prosedur Operasi Progress Penagihan dan Pemeriksaan Hasil Pekerjaan (PO-PRJ-15)

5.     Tanggung Jawab

5.1.      Manager Keuangan  bertanggung jawab terhadap terlaksananya  kesepakatan data final account proyek.

5.2.     Project Manager  bertanggung jawab terhadap laporan data final account proyek sesuai dengan status proyek bersangkutan.

5.3.     Site Manager bertanggung jawab terhadap inputan akurasi data data final account proyek.

5.4.     Project Control bertanggung jawab terhadap inputan akurasi data final account proyek

6.     Uraian Proses

Catatan :

Pada prinsipnya pelaksanaan final  account disesuaikan dengan status proyek, antara lain : status progress 100%, status tutup tahun anggaran dan status proyek closing. Proyek Closing dilaksanakan setelah “serah terima  ke dua” / Final Handover.

6.1.      Manager Keuangan mengajukan permintaan kepada Project Manager untuk melakukan final account proyek sesuai dengan status proyek bersangkutan, antara lain : status progress proyek 100%, status tutup tahun anggaran atau status proyek closing.

6.2.     Project Manager sesuai permintaan final account menyepakati jadwal koordinasi dengan Departemen Keuangan dalam upaya melakukan klarifikasi dan kesepakatan data final account.

6.3.     Project Manager sesuai jadwal meminta kepada Site manager, Project Controller dan Quantity Surveyor, bersama-sama melakukan klarifikasi data-data final account dengan Departemen Keuangan antara lain sesuai ” Template Final Account ” :

a.      Project Performance (FA-01)

b.     Performance Profit & Loss (FA-02)

c.      Budget Review (FA-03)

d.     Finance Project Performance (FA-04)

e.      Certificate Payment Status (FA-05)

f.       Performance Indek Kepuasan Pelanggan (FA-06) – file Markom

g.      Analisis And Corrective Action (FA-07 )

h.     Minutes of Meeting / MoM (FA-08)

catatan :

Data Indeks Kepuasan Pelanggan dikoordinasikan dengan Marketing Komunikasi sesuai dengan respon Quistioner.

6.4.    Project Manager sesuai klarifikasi data final account, membuat undangan kepada Tim Core 3 dan Steering Group dalam rangka presentasi final account proyek.

6.5.      Project Manager sesuai jadwal melakukan presentasi final account proyek kepada Steering Group sesuai ” Template Final Account ” (SP-PRJ-11-01)

6.6.    Project Manager menerima saran dan arahan dari Steering group serta mencatatnya dalam format Minutes Of Meeting (FA-09) untuk ditindaklanjuti oleh para pihak , sesuai dengan tanggung jawabnya.

catatan :

Hasil presentasi final account bila telah mendapat persetujuan Direksi, khususnya laporan Project Performance (FA-01), laporan Performance Profit & Loss (FA-02) dan Minutes Of Meeting / MOM (FA-08) didistribusikan kepada Manager Keuangan.
