Alokasi TAMBAH / KURANGKAN UNCOMMITTED COST

DEFINISI

a.     Alokasi ini dipergunakan untuk saving cost, jadi proses saving cost harus dilakukan secara rutin.

Untuk itu  sejak dari awal pekerjaan harus segera melakukan pembuatan commitment, seperti kontrak dan PO, dan segera lakukan pengurangan uncommitted cost, dengan demikian di awal sudah bisa prediksi apakah budget kita bisa dilakukan penghematan atau tidak (terlihat pergerakan nilai dari Projected Cost akan lebih kecil dari Current Budget)

b.    Alokasi ini dipergunakan juga apabila dalam perjalanan pelaksanaan pekerjaan, ternyata memerlukan link budget terhadap suatu budget code yang nilai Uncommitment (sisa budget yang belum dilakukan kesepakatan) kurang atau habis karena sudah terpakai oleh commitment (dalam bentuk PO/Kontrak/Indirect Cost)

Untuk itu perlu ditambahkan nilai Uncommitted Cost tersebut, agar transaksi bisa tetap di proses.

Setiap penambahan atau pengurangan Uncommitment Cost, akan mempengaruhi nilai Over terhadap Projected Cost, maka perlu diperhatikan langkah-langkah sebagai berikut:

b.1.        Jangan dipaksakan untuk melakukan link budget untuk budget code yang akan digunakan pada item PO/Kontrak/Indirect Cost tersebut jika nilai dari item tersebut masih lebih besar dari nilai Uncommitted Cost yang tersedia.

Walau masih bisa di save, tetap jika dipaksakan, saat ditambahkan uncommitted cost, maka harus di link ulang, sehingga menjadi tidak efisien

Jadi tambahkan terlebih dahulu nilai Uncommitted Cost didalam budget code yang diperlukan sebelum melakukan transaksi, agar saat dilakukan proses, sudah bisa di tanda-tangan.

b.2.        Lakukan pengurangan dari uncommitted existing (saving cost) sebelum melakukan penambahan, hal ini agar tidak menimbulkan over budget

b.3.        Lakukan penambahan uncommitted cost jika memang sudah tidak ada lagi budget code lain yang bisa dilakukan saving cost

c.     Dalam hal persetujuan perubahan Uncommitted cost (akan merubah nilai Projected Cost), maka opsi persetujuan adalah sebagai berikut:

Jika perubahan nilai Uncommitted menyebabkan “nilai Current Under Over tetap positif (+)”, maka validasi perubahan budget code tersebut adalah:

ü Dibuat oleh          : Quantity Surveyor

ü Diperiksa oleh     : Project Control

ü Disetujui oleh      : Project Manager

d.    Prosentasi profit diperoleh dari rumus:

Rencana Profit = (Nilai RAB – Original Budget) / Original RAB x 100%

Realisasi Profit = (Current RAB – Projected Cost) / Current RAB x 100%

d.1.        Jika perubahan nilai Uncommitted menyebabkan “nilai Current Under Over negatif (-)”, atau kondisi “100% ≤ nilai Realisasi Profit / Rencana Profit nilainya > 20%” maka validasi perubahan budget code tersebut adalah:

ü Dibuat oleh          : Quantity Surveyor

ü Diperiksa oleh     : Project Manager

ü Disetujui oleh      : SPV

d.2.        Jika perubahan nilai Uncommitted menyebabkan “nilai Realisasi Profit / Rencana Profit nilainya ≤ 20%” maka validasi perubahan budget code tersebut adalah:

ü Dibuat oleh          : Quantity Surveyor

ü Diperiksa oleh     : Project Manager

ü Disetujui oleh      : Direksi
